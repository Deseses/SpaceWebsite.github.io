Module: textcart

Модуль для обрезки длинных текстов в карточках астероидов и комет Обрезает текст описания до указанной длины и добавляет многоточие

Methods

(inner) truncate(str, maxlength) → {string}

Обрезает строку до указанной максимальной длины
Parameters:
Name Type Description
str string Исходная строка для обрезки
maxlength number Максимальная допустимая длина строки
Source:
  • textcart.js, line 7
Returns:
Обрезанная строка с добавлением '…' в конце, если исходная строка превышает maxlength, иначе возвращает исходную строку
Type
string
Example
truncate("Очень длинный текст описания астероида", 20)
// Возвращает: "Очень длинный текс…"
