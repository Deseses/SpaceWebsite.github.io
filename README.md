## 🚀 Обзор проекта

Проект представляет собой образовательный веб-сайт, посвященный малым телам Солнечной системы. Сайт содержит информационные страницы об астероидах и кометах, интерактивную 3D-модель Солнечной системы, 3D-модели объектов, звездное небо с эффектом телескопа, а также систему регистрации с CAPTCHA.

### Основные страницы

| Страница | Файл | Описание |
|----------|------|----------|
| Главная | `index.html` | Быстрые факты, видео, список научных книг |
| Астероиды | `asteroid.html` | Информация об астероидах + 3D-модель |
| Кометы | `kometa.html` | Информация о кометах + 3D-модель |
| Солнечная система | `solar-system.html` | Интерактивная 3D-модель с планетами |
| Регистрация | `regist.html` | Форма регистрации с CAPTCHA |

## ✨ Функциональные возможности

### 🎨 Визуальные эффекты
- **Звездное небо с параллаксом** — фон из мерцающих звезд, реагирующий на прокрутку
- **Анимированный фон** — динамическое звездное поле с мерцанием (CSS-анимация)

### 🪐 3D-визуализации
| Компонент | Описание |
|-----------|----------|
| **Астероид** | Вращающаяся 3D-модель с управлением камерой |
| **Комета** | 3D-модель ледяного ядра кометы |
| **Солнечная система** | Полная 3D-модель: Солнце, 8 планет с орбитами, пояс астероидов, Облако Оорта |

### 🔭 Интерактивные элементы
- **Телескоп/лупа** — приближение звездного неба под курсором с регулируемым зумом (0.5x–8x)
- **Карточки с hover-эффектом** — всплывающие описания астероидов и комет
- **Пагинация книг** — постраничная загрузка списка научно-популярной литературы (по 3 книги)

### 📱 Адаптивный дизайн
- **Бургер-меню** для мобильных устройств (ширина < 500px)
- **Адаптивная сетка** CSS Grid и Flexbox

### 🔐 Форма регистрации
- Валидация полей (имя, фамилия, email, телефон, пароль)
- **CAPTCHA** — текстовая и математическая с автоматическим переключением
- Подтверждение пароля
- Принятие пользовательского соглашения

## 📁 Структура проекта
proect/
- index.html # Главная страница
- asteroid.html # Страница астероидов
- kometa.html # Страница комет
- solar-system.html # 3D-модель Солнечной системы
- regist.html # Форма регистрации
- style.css # Основные стили
- regist-style.css # Стили формы регистрации
- package.json # Зависимости и скрипты
- package-lock.json # Фиксация версий
- codes/ # JavaScript модули (8 файлов)
    - 3dasteroid.js # 3D модель астероида
    - 3dkomet.js # 3D модель кометы
    - solar-system.js # 3D модель Солнечной системы
    - stars.js # Звездный фон с параллаксом
    - telescope.js # Интерактивный телескоп
    - book.js # Пагинация книг
    - textcart.js # Обрезка текста карточек
    - capcha.js # CAPTCHA проверка
- models/ # 3D-модели (GLTF)
    - asteroid/ # Модель астероида
        - scene.gltf
    - iceblock/ # Модель кометы
        - scene.gltf
- img/ # Изображения (логотип, планеты, кометы)
- video/ # Видеофайлы для фоновой анимации
-  docs/ # Сгенерированная JSDoc документация
